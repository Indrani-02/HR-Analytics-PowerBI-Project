# HR Analytics Dashboard using Power BI

## Project Overview
This project is an HR Analytics Dashboard created using Power BI to analyze employee attrition, salary trends, work-life balance, and department-wise performance.

The dashboard provides interactive visualizations and valuable HR insights that help in better workforce decision-making.

---

## Tools & Technologies Used
- Power BI
- Excel
- Power Query
- DAX

---

## Key KPIs
- Average Attrition Rate for All Departments
- Average Hourly Rate of Male Research Scientists
- Attrition Rate vs Monthly Income
- Average Working Years for Each Department
- Job Role vs Work-Life Balance
- Attrition Rate vs Years Since Last Promotion

---

## Dashboard Features
- Interactive Slicers
- Dynamic KPI Cards
- Department-wise Analysis
- Attrition Insights
- Employee Trend Analysis
- Professional Data Visualizations

This dashboard helps HR teams understand employee attrition patterns, employee satisfaction, and workforce trends to support data-driven decisions.
